Toolkit Help & Documentation

Getting Started

The HR Operations Toolkit provides six essential tools to streamline your daily HR tasks. Click on any tool card to access its functionality.

Quick Tips:

  • Use the search bar to find tools quickly
  • Click the help icon for tool-specific guidance
  • Dark mode is available for comfortable viewing
  • All files are processed securely on our servers

File Guidelines:

  • Max file size: 16MB per upload
  • Supported image formats: PNG, JPG, JPEG
  • PDF processing available for PDF documents
  • Bulk uploads supported via ZIP files

Tool Descriptions

ID Image Processor

Crop and optimize candidate images for ID cards and HRMS. Supports both single and bulk processing.

Complete PDF Toolkit

Merge, split, compress, add watermarks, and convert PDF files. All processing done offline.

Employee Exit Verifier

Instantly check candidate exit status and Full & Final Settlement details from HR database.

Candidate Experience Calculator

Analyze and calculate total work experience from previous companies and certificates.

Offer Release Tracker

Track offer releases with timestamps, status updates, and candidate communication logs.

Employee ID Availability

Check and generate available employee IDs following company patterns and allocation rules.

Security & Privacy

All uploaded files are processed securely and automatically deleted after 24 hours. No personal data is stored permanently. This toolkit complies with company data security policies.
